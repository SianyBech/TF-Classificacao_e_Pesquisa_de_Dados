DESCRIÇÃO DO PROBLEMA / APLICATIVO:
Nosso projeto irá utilizar a base de dados do governo do Rio Grande do Sul para fazermos
um catálogo de matrículas (divididas por sexo), feitas nos municípios gaúchos entre os
anos de 2014 e 2024 (período em que há o fornecimento de dados). A partir deste
catálogo, informaremos ao usuário (após este escrever um município e indicar um
período), se houve um aumento ou diminuição no número de matrículas e de quanto foi
este número. É um problema relevante, pois, além de a educação ser um tópico
extremamente necessário, com o período de dados fornecido (2014 – 2024), é possível
relacionar estes números a pandemia e/ou a enchente, além de podermos destacar outras
diferenças sociais entre cidades maiores e cidades menores. Com este projeto, esperamos
oferecer ao usuário, uma base sólida e direta a respeito destes índices, sem a necessidade
de ficar pesquisando estes números pelos sites do governo (ou dos municípios), já que
nossos dados possuem fonte direta de lá.
FUNCIONALIDADES PREVISTAS:
1. Usuário pode informar a cidade desejada (por nome e também tentaremos colocar
pesquisa pelo número da cidade pelo IBGE);
2. Usuário pode informar o período a ser procurado (indicaria 2 anos para nossa
comparação);
3. Usuário pode informar qual sexo ele deseja (masculino/feminino) para ter
conhecimento destes dados;
4. A partir disso, nosso programa informará se houve um aumento ou diminuição de
matrículas, e de quanto foi este valor;
5. Se possível, também iremos incluir busca por idade e cor!
